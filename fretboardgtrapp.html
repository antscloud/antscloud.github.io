Site Web

Afin d'apprendre un peu plus le développement web mais aussi et surtout pour apprendre le déploiement d'une application Python sur un serveur j'ai décidé de coder un site web pour avoir une interface graphique du projet de fretboardgtr. Pour cela j'ai décidé d'utiliser Flask pour la partie serveur.

Fonctionnement général

Quand les différents paramètres sont remplis sur le site, un objet JSON est à chaque fois modifié. Par exemple quand on change une note dans le panneau de gauche, l'attribut "notes" de l'objet va être modifié. Et ceci pour chaque configuration. Quand on appuie sur le bouton "generate", une requête contenant cet objet est envoyé via une méthode POST. Lorsque le serveur la reçoit il va générer un diagramme en fonction de ces paramètres et enregistrer l'image dans un dossier particulier. Il renvoie alors une chaîne de caractères contenant un url. Cette url est une autre route qui permet de lire l'image sur le site. Chaque image est stocké avec un nom unique (uuid). A chaque requête Flask analyse si il y a plus de 100 images. Si tel est le cas, il en supprime 50 en commençant par les plus vieilles. Ainsi le nombre maximum d'image stocké sur le serveur est de 100 et permet de ne pas surcharger le stockage.
